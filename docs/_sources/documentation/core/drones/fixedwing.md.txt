# Fixedwing

## Model Description

Small tube-and-wing fixed wing UAV with a single motor (~2.5 Kg).

## Control Mode

Only one control mode is provided.
Setpoints are mapped directly to `[pitch, roll, yaw, thrust]` actuator commands.
